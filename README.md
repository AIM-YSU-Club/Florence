# Florence (FE)

## How to Run our project?

### 1. Clone our repository
```zsh
git clone https://github.com/AIM-YSU-Club/Florence.git
```

### 2. Setup packages
#### (Default) Yarn
```zsh
yarn install
```

#### npm
```zsh
npm install
```

### 3. Run Project
#### (Default) yarn
```zsh
yarn dev
```
#### npm
```zsh
npm run dev
```
